# OIBSIP
# Online Examination System
A Java based Online Examination System developed as part of the Oasis Infobyte Java Development Internship. The application provides a GUI for students to log in,
update their profile, take a timed multiple-choice examination, and view their results.
## Features
  ### User Authentication
  - User Login
  - Username & Password Validation
  ### Profile Management
  - Update Display Name
  - Start Examination
  ### Examination Module
  - Multiple Choice Questions (MCQs)
  - Four Options for Each Question
  - Previous & Next Navigation
  - Save Selected Answers
  - Countdown Timer (30 Minutes)
  - Automatic Submission when Time Expires
  - Manual Submission with Confirmation Dialog
  ### Result Module
  - Total Score
  - Percentage
  - Time Taken
  - Logout Functionality
  ### Additional Features
  - Window Close Confirmation During Exam
  - Object-Oriented Programming (OOP) Design
  - Easy-to-Understand GUI
## 🛠️ Technologies Used

## How to Run
1. Clone or download this repository.
2. Open the project in Eclipse, IntelliJ IDEA, or VS Code.
3. Compile all Java files.
4. Run `LibraryMain.java`.
5. Login using the default credentials.
